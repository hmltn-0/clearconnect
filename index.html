clearconnect

make way for the universal internet

How it works.

  1. You have a wireless device. We will have a library of community-developed open source hardware schemata and network interface protocols. This is very similar to the Linux movement.
  2. You install the open-source firmware we will build. You need control over the hardware's network devices in order for this to work.
  3. You install the open-source software that runs the protocol.
  4. You are now contributing to the network. A blockchain protocol pays you in a token for your proof-of-work.

The internet is a topos.
